# 项目初始化

```bash
bun create expo <project-name>
```

连接真机(android)后, 开发模式运行

```bash
bun run android
```

发布 Release 版本到真机(android)
```bash
bunx expo run:android --variant Release
```

## 测试热更新

因为本地开发没有 http, 所以先在 `app.json` 里配置

```json
"updates": {
  "enabled": true,
  "url": "http://192.168.8.114:6688/manifests/ch_i2h8yaNCz8kNRxKTLRzfPK5RvJ2NCjJh",
  "fallbackToCacheTimeout": 0

```

本地 IP 的获取

```bash
# 使用 ipconfig
ipconfig getifaddr en0
# 或
ifconfig | grep "inet " | grep -v 127.0.0.1
```
