# 热更新工作流 - 获取渠道信息

## 查询渠道

```bash
# 根据渠道 key 查询渠道信息
curl -s -X GET "http://192.168.8.114:6688/trpc/hotUpdate.manage.channels.byKey?input=%7B%22channelKey%22%3A%22prod_demo_app_channel_key_12345678%22%7D" \
  -H "Authorization: Bearer $ACCESS_TOKEN" | jq '.result.data'
```

## URL 编码说明

URL 参数 `input` 需要经过编码:
- 原始: `{"channelKey":"prod_demo_app_channel_key_12345678"}`
- 编码后: `%7B%22channelKey%22%3A%22prod_demo_app_channel_key_12345678%22%7D`

可以使用在线工具或命令编码:
```bash
echo '{"channelKey":"prod_demo_app_channel_key_12345678"}' | jq -sRr @uri
```
