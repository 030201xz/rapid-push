# 热更新工作流 - 管理员登录

## 登录获取 Token

```bash
# 登录并获取 access token
curl -X POST "http://192.168.8.114:6688/trpc/core.identify.auth.login" \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"Admin@123456"}' | jq

# 提取 token 到环境变量
export ACCESS_TOKEN=$(curl -s -X POST "http://192.168.8.114:6688/trpc/core.identify.auth.login" \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"Admin@123456"}' | jq -r '.result.data.accessToken')

# 验证 token
echo "Token: $ACCESS_TOKEN"
```


## 注意事项

- Token 有效期为 7 天
- 登录后需要保存 `ACCESS_TOKEN` 环境变量供后续步骤使用
